# Oryxen Frontend

This repo contains the frontend for Oryxen, a sophisticated fitness platform
for training programs, exercise progressions, class scheduling, and nutrition
planning.

## Structure

- apps/frontend: Next.js app
- tests: Frontend test assets and documentation

## Quick start

- Frontend dev server: npm run dev:frontend

## Notes

- The frontend app manages its own dependencies in apps/frontend.
- Add frontend tests under tests/ and app-level test folders as they are introduced.
