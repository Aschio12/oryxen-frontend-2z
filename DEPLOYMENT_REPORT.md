## Animation System (40+ New Utilities)

### Glassmorphism Effects
- `.glass` - Standard glassmorphism with blur
- `.glass-soft` - Subtle glass effect
- `.glass-premium` - Premium gradient glass

### Motion Utilities
- `.motion-rotate-in` - Rotate + scale entry
- `.motion-bounce-in` - Bouncing entry animation
- `.motion-shimmer` - Shimmer effect
- `.motion-glow-pulse` - Glowing pulse animation
- `.motion-slide-up` - Upward slide entry
- `.motion-slide-down` - Downward slide entry
- `.motion-fade-in-scale` - Fade + scale combination
- `.motion-stagger-1` through `.motion-stagger-5` - Staggered sequence delays

### Hover Effects
- `.hover-lift` - Lift on hover with shadow
- `.hover-scale-gold` - Scale with gold glow
- `.hover-glass` - Enhanced glass effect
- `.hover-text-glow` - Text glow effect

- `.transition-fast` - 0.3s fast transition
